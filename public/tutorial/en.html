SRT Player Tutorial

0. What's this tool for

This tool is designed for language learners. Its has 2 main features: separate subtitle and waveform.

Separate subtitle means you can copy and paste to look up new words.

Waveform helps practising listening.

1. Import files

  • What to import
    • video/audio files (appears on left)
    • subtitle files (appears on right)
  • Supported format
    • Video and audio depends on your browser, you can check it here.
    • Subtitle: srt, ssa, ass
  • If you import multiple videos and subtitles, their orders need to match. You can drag items to reorder.
  • It's OK to import only one video/audio

2. Subtitle

Check the subtitle settings panel. (press B and it will show up)

Videos and subtitles found on Internet may not match time-wise, and you'll have to adjust subtitle delay. This can be easily done by clicking on the timestamp like 00:00:44,478 or 00:00:46,713, and delay will be adjusted to the current video time. If delay is set, it will be shown under the cc icon in red on top right.

3. Waveform

Generating waveform

Press S to bring up waveform control.

By default waveform is disabled. To enable waveform, there are 2 options:

  1. Generate from the current video/audio file
  2. Generate from a separate file

Most of the time option 1 is enough. But for technical reasons, the generated waveform might not be accurate, in which case you'll have to choose option 2 and feed browser a separate file with format that's easier to digest.

For example, you can convert the video file to a mp3 file, and use that mp3 file to generate waveform.

Interact with waveform

Click on the waveform will:

  1. set video to the corresponding time
  2. set a replay location, indicated by a yellow line

Press R or click the replay icon(bottom right of video) to replay from the previously set location
